# Autonomous-Backup-Orchestrator-ABO-
Autonomous Backup Orchestrator (ABO) — это прототип системы интеллектуальной оркестрации резервного копирования, разработанный для защиты данных от атак типа Ransomware (вирусов-шифровщиков).  Система реализует стратегию 3-2-1 и обеспечивает изоляцию хранилищ через зашифрованные SSH-тоннели.
